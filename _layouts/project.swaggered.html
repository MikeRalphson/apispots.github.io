swagger.ed

A Chrome extension for APIs visualization

swagger.ed is a simple Chrome browser extension that displays a visual representation of an API modelled using the OpenAPI spec (fka Swagger). Simply add the extension on your browser directly from the Web Store and visualize API definitions directly from the raw JSON / YAML format.

If you need some real life API specs to have a go with the extension, have a look at the fantastic API Stack catalogue by the API Evangelist (Kin Lane) or the APIs Guru directory. All registered APIs come with an OpenAPI spec that you can open and visualize using the add-on.
